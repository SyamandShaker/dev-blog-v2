VS Code

Visual Studio Code (VS Code) is the name of the latest developer tool from Microsoft.
By name, you might think it is the latest edition of Visual Studio, but VS Code is more like its flexible little brother: A lean code editor that supports software developers not only in editing, but also in managing and testing code.

GIT-SCM

Git is a distributed version control system that allows you to track changes in source code during software development.
It is designed to coordinate the work of programmers, but it can be used to track changes in any set of files. Its goals are speed, data integrity, and support for distributed and non-linear workflows.
